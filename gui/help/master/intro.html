Phycas

The Phycas Software consists of two components – a backend Processor and a front-end Graphical User Interface (GUI).  The Phycas software is used to perform of a series of commands, each with its own options.  These commands are entered using the Nexus format.

Using the Phycas GUI

The GUI consists of two windows – the main window and the output window. 

Main Window

When the GUI is initially launched, the welcome screen is displayed in the main window.    This screen provides a text field for the user to enter a command directly.  Commands can also be generated using its screen.  Each command has a menu item on one of the menus in the menu bar of the main window.  When a menu item is selected, the main window displays the appropriate command’s screen. When the OK button is pressed on that screen, the Nexus command string is generated and sent to the Processor.  The main window then switches back to the welcome screen.  If an error occurs in the Processor or more information is needed from the user, the main window will switch back to the last command screen viewed.  Additional menu items in the menu bar allow the user to show the output window or the help window or the welcome screen, or to exit Phycas.  The bottom of the main window contains the Message Area.  When a field on a command screen has an error, its label will turn red and a message will be displayed in the Message Area.

Output Window

The output window initially contains a single tab labeled output.  All output from the Processor is written there.  If a command is issued by the user containing data that needs to be plotted, tabs will be added to this window to allow the user to generate trace plots of the data returned by the Processor.  The output window can be closed without exiting Phycas.  The menu bar of this window contains options for saving and clearing the output text, and for printing, saving, closing, and editing the properties of graphs.  Most of these actions can also be performed using a popup menu, which is made visible by right-clicking the mouse in either the output text area or on a graph.
